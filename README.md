# AI Task Assistant

A smart productivity tool built with React, TypeScript, SCSS, and Vite.

## Overview

AI Task Assistant is a simple but powerful to-do app designed to help you stay organized.  
You can add, edit, delete, and complete tasks, all styled with SCSS and saved in your browser.  
The app also integrates AI features for smarter task management.

## Live Demo

[View it here](https://ai-powered-task-assistant.vercel.app/)

## Features

- Add, edit, and delete tasks
- Tasks are saved in your browser
- Smooth, clean UI styled with SCSS
- Light/dark theme toggle
- Filter tasks by status, category, or sort order
- AI-powered task rephrasing
- AI-powered automatic categorization of tasks
- Color-coded category badges for quick identification:
  - **Work**: Blue
  - **Personal**: Green
  - **Home**: Teal
  - **Health**: Yellow
  - **Errands**: Orange
  - **Study**: Purple
  - **Other**: Gray
